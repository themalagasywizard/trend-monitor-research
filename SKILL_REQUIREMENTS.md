# Trend Monitor - Skill Requirements

## Core Capabilities Needed

### Current Tools (Available)
- ✅ `web_search` - Brave search API (needs API key)
- ✅ `web_fetch` - Fetch specific URLs
- ✅ `cron` - Schedule twice-daily runs
- ✅ `write` - Save summaries and ideas

### Recommended Skills to Install

From clawhub.com, search for:

1. **Social Media Aggregation**
   - For better X/Twitter scraping
   - Reddit API integration
   - Hacker News monitoring

2. **Content Analysis**
   - Sentiment analysis
   - Trend detection
   - Topic clustering

3. **RSS/Feed Monitoring**
   - Tech blog feeds
   - Product Hunt
   - Indie Hackers

4. **Newsletter Monitoring**
   - TLDR Tech
   - Morning Brew Tech
   - Stratechery (if accessible)

## To Install Skills

```bash
openclaw skill search "social media"
openclaw skill search "rss feed"
openclaw skill search "sentiment analysis"
```

Then install with:
```bash
openclaw skill install <skill-name>
```
